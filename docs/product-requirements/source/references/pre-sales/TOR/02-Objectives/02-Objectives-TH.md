# วัตถุประสงค์ (Objectives)

2.1 เพื่อศึกษา วิเคราะห์ ออกแบบ ฐานข้อมูลต่าง ๆ ที่เกี่ยวข้อง
2.2 เพื่อรวบรวมข้อมูลที่สำคัญซึ่งอยู่ภายใต้ภารกิจงานของสำนักบริหารบุคคลให้อยู่บนระบบฐานข้อมูลเดียวกัน
2.3 เพื่อพัฒนาโปรแกรมระบบข้อมูลเพื่อให้สามารถประมวลผลและแสดงผลตามความต้องการของผู้ใช้ เพื่อใช้ในการปฏิบัติงานและบริหารงานด้านทรัพยากรบุคคลในภาพรวมของกระทรวงการต่างประเทศ
2.4 เพื่อวางโครงสร้างพื้นฐานให้พร้อมรองรับการขยายขอบเขตให้ครอบคลุมข้อมูลทั้งหมดของสำนักบริหารบุคคล กระทรวงการต่างประเทศ
